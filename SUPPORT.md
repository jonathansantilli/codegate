# Support

## Usage Questions

For usage help, open a GitHub Discussion (preferred) or a GitHub issue with steps to reproduce.

## Bug Reports

Use the bug report issue template and include:

- command used
- expected result
- actual result
- environment (OS, Node version)

## Security Reports

For security vulnerabilities, do not use public issues.
Follow [SECURITY.md](./SECURITY.md).

## Maintainer Contact

General maintainer contact: `maintainers@codegate.dev`
